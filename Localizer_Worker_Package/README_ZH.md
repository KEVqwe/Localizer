# 🚀 Localizer 远程分布式节点部署指南 (v1.0)

欢迎加入 Localizer 算力集群！通过此部署包，你可以让你的电脑成为视频本地化处理的一个“Worker 节点”。

## 📦 准备工作

在开始之前，请确保你已经安装了以下软件（已包含在自动安装脚本中，但建议手动确认）：
1. **Miniconda** (Python 环境管理)
2. **Node.js** (用于运行 PM2 进程管理器)
3. **FFmpeg** (必备多媒体引擎)

---

## 🛠️ 第一步：环境安装 (只需执行一次)

双击运行目录下的 **`INSTALL.bat`**。
*   它会为你自动创建一个名为 `py311` 的 Python 沙盒环境。
*   它会安装所有的 AI 模型组件（Torch, WhisperX 等）。
*   **注意**：此步骤可能需要 10-15 分钟，具体取决于你的网速（正在下载约 2GB 的 AI 模型）。

---

## 🏃 第二步：启动节点

双击运行目录下的 **`START_WORKER.bat`**。
*   **首次运行**：它会自动创建一个 `.env` 配置文件。
*   **配置确认**：如果这是你第一次运行，请确认你的 Master 电脑（主控电脑）的 IP 已经填入了 `.env` 中。
*   **启动服务**：脚本会自动启动 PM2 背景进程。看到“启动成功”提示后，你就可以关闭这个窗口了。

---

## 🔍 第三步：检查状态

如果你在 Master 电脑的 Flower 界面上看不到自己，请运行：
**`DIAGNOSTIC_TEST.bat`**
*   它会全自动检测你的网络配置、防火墙设置以及环境是否完整。
*   根据脚本的“红色”失败提示进行排查即可。

---

## ❓ 常见问题排查

1.  **窗口运行到一半闪退？**
    *   检查是否安装了本地 Python 冲突，建议卸载其他 Python 只保留 Miniconda。
2.  **Flower 看不到我？**
    *   通常是防火墙拦截了 6379 端口。请确认 Master 电脑已开放防火墙策略。
3.  **报错 `ImportError`？**
    *   说明依赖没装全，请重新运行 `INSTALL.bat`。

---

祝你部署顺利！如果有任何报错请联系 Master PC 负责人。
